# Visual Events Website

## 🌟 Features
- **Home Page:** Welcome users with an engaging landing page.
- **About Us:** Learn about our journey and commitment.
- **Services:** Explore the various event planning services we offer.
- **Gallery:** View stunning images from past events.
- **Contact Us:** Reach out with inquiries or get in touch.
- **Booking Form:** Easily book your event with our user-friendly form.
- **Login and Registration:** Secure client access.

## 🚀 Getting Started
1. **Clone the Repository:**
    ```bash
    git clone https://github.com/savidhps/Visual-Events.git
    ```
2. **Navigate to the Project Directory:**
    ```bash
    cd Visual-Events
    ```
3. **Open in Browser:**
    - Simply open `index.html` in your preferred browser.
